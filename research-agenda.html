Research Gaps & Paper Agenda

The central gap is access-adaptive, evidence-calibrated auditing for multi-teacher, multi-role, multi-stage pipelines—with abstention and false-attribution control.

Highest-value unresolved problem

Recover a calibrated set of candidate teachers and their roles from a model trained through a multi-stage supply chain, while adapting to the auditor’s available access.

Output-only screening. Select prompts where candidate teachers disagree in decisions, preferences, critiques, safety boundaries, or tool policies.
Reference residualization. When a base or earlier checkpoint exists, measure what changed rather than raw similarity.
Role-conditioned signatures. Separate generator, reasoner, judge, critic, filter, safety, and tool-use evidence.
Evidence escalation. Add top-k probabilities, logits, representations, routing, and weight trajectories when access permits.
Open-world inference. Return a teacher set or abstain; never force the closest known model to be the source.

Strong novelty

  • Multi-teacher and multi-role attribution
  • Unified black-box to white-box method
  • Unknown-teacher and abstention support
  • False-attribution control
  • Public-artifact + behavioral + internal evidence fusion

Avoid

  • Style-only attribution
  • Benchmark-score similarity as provenance
  • Unique-teacher claims without an unknown class
  • Conflating weight inheritance with response distillation
  • Treating provider allegations as experimental ground truth

Candidate framing

DistillGraph: Evidence-Calibrated Multi-Teacher Distillation Lineage Auditing for Large Language Models.

Input: suspect model, candidate teachers, optional reference checkpoints, and an access budget. Output: teacher set, role labels, evidence channels, calibrated confidence, and abstention.
